## Styling Patterns Used

### Headers
```html
<h2 class="border-4 border-black inline-block px-6 py-3 bg-white">
  [ SECTION_TITLE ]
</h2>
```

### Content Boxes
```html
<div class="bg-white border-4 border-black p-6 hover:shadow-[8px_8px_0px_0px_rgba(0,0,0,1)]">
  <p>&gt; Content here</p>
</div>
```

### Lists
```html
<li>&gt; List item</li>
```

### Comments
```html
<p class="text-gray-600">// This is a comment</p>
```
